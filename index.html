WeChat RSS Feed

This project provides an RSS feed for WeChat official accounts. It allows users to subscribe to WeChat updates using any RSS reader.

Features

  • Fetch updates from WeChat official accounts
  • Generate RSS feeds
  • Easy to set up and use

Getting Started

Prerequisites

  • Python 3.x
  • pip

Installation

git clone https://github.com/michael180831/wechat-rss-feed.git
cd wechat-rss-feed
pip install -r requirements.txt

Usage

python main.py
